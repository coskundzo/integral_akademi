# İntegral Akademi - Matematik Soru Çözüm Platformu

Modern ve responsive bir matematik eğitim platformu web sitesi.

## 🎯 Özellikler

- ✅ Matematik Sorularının Detaylı Çözümleri
- ✅ Pratik Yollar & Püf Noktalar
- ✅ ÖSYM Mantığına Uygun Anlatımlar
- 📱 Tamamen Responsive Tasarım
- 🎨 Modern ve Kullanıcı Dostu Arayüz
- ⚡ Hızlı ve Optimize Edilmiş
- 🔔 Bildirim Desteği

## 📌 Amaç

👉 Soruyu Sadece Çözmek Değil, Mantığını Kavratmak
👉 Çözümler Eğitim Amaçlıdır

## 🚀 Kurulum

1. Projeyi klonlayın veya indirin
2. `index.html` dosyasını bir web tarayıcısında açın
3. Herhangi bir sunucuya yüklemek için tüm dosyaları sunucunuza yükleyin

## 📁 Dosya Yapısı

```
integral_akademi/
│
├── index.html          # Ana HTML dosyası
├── style.css           # CSS stilleri
├── script.js           # JavaScript dosyası
└── README.md           # Proje açıklaması
```

## 🛠️ Teknolojiler

- HTML5
- CSS3 (Flexbox, Grid, Animations)
- JavaScript (ES6+)
- Font Awesome Icons

## 🎨 Tasarım Özellikleri

- Modern gradient tasarım
- Smooth scroll efektleri
- Hover animasyonları
- Responsive grid layout
- Mobile-first yaklaşım
- Intersection Observer API ile performans optimizasyonu
